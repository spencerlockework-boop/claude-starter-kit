# Brand Design References

When the user asks to match a specific brand's aesthetic, check the `brands/` subdirectory for a matching DESIGN.md file.

Each brand file contains:
- Visual theme and color palette (exact tokens)
- Typography scale and font choices
- Component styling (buttons, cards, inputs, navigation)
- Layout principles and spacing
- Elevation and depth patterns
- Responsive behavior
- Guardrails (what NOT to do)

Available brands: check `brands/` directory for current list.

If no brand file exists for the requested brand, use the closest match as a starting point and note what you're adapting.
